## ðŸ§  Key Principles Remembered

Every project in this repo adheres to the "Modern Android Development" (MAD) scorecard:

* **Declarative UI:** No more XML; the UI is a function of the state.
* **State Hoisting:** Moving state to a caller to make composables stateless and easier to test.
* **Unidirectional Data Flow:** Events flow up, state flows down.
